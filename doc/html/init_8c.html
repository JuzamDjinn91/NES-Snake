§ calc_random_item_position()

void calc_random_item_position ( void  )

This function calculates the coordinates of an grow-item. It stores the calculated coordinates into global fields 'coord_x' and 'coord_y'. In terms of the game structure, this function should be placed in file 'update.c'. But since I would like to spawn the initial items randomly as well, this function needs to placed in file 'init.c'.
