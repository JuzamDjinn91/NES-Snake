§ MAPARRAY_ADR

#define MAPARRAY_ADR (   x,
  y 
)    ((y<<2)|(x>>3))

Macro for calculating in which tile of the 32*30 tiles the given position is placed. Optimized with bitshifting, arithmetic pendant is (((y/8)*32)+(x/8)). x and y are assumed to be Sprite-coordinates (not Tile-coordinates).
